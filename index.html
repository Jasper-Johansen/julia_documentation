Introduction

Julia is a high-level, high-performance programming language designed for technical computing. Launched in 2012, it combines the ease of use of scripting languages with the speed of compiled languages, making it ideal for numerical analysis and computational science. Its rich ecosystem supports parallel and distributed computing seamlessly.

With an intuitive syntax reminiscent of Python and MATLAB, Julia enables rapid prototyping and efficient execution. It features powerful libraries for data manipulation, visualization, and machine learning, fostering a vibrant community of users and contributors. Julia's ability to handle large datasets and perform complex mathematical computations positions it as a go-to choice for researchers and data scientists alike.

  • Free and open-source
  • Extensive community support
  • User-defined types are as fast and compact as built-ins
  • No need to vectorize code for performance
  • Designed for parallelism and distributed computation
Getting Started

The easiest way to learn and experiment with Julia is by starting an interactive session (also known as a read-eval-print loop or "REPL") by double-clicking the Julia executable or running julia from the command line, after installing it from Julia Download Instructions.

julia> 1 + 2
3
Comments

Julia syntax is simple and easy to learn, making it an excellent language for beginners in statistical computing.

Comments in Julia are created using the hash symbol `#`. Anything following this symbol on the same line is ignored by the interpreter. And for multi-line comments, use "#= ...=#"

# This is a single-line comment

#=
This is a multi-line comment.
It can span multiple lines.
=#
Variables and Constants

Julia provides an extremely flexible system for naming variables. Variable names are case-sensitive, and have no semantic meaning (that is, the language will not treat variables differently based on their names).


julia> y = -3
-3

julia> Z = "My string"
"My string"

julia> customary_phrase = "Hello world!"
"Hello world!"

julia> UniversalDeclarationOfHumanRightsStart = "人人生而自由，在尊严和权利上一律平等。"
"人人生而自由，在尊严和权利上一律平等。"

Constants, on the other hand, hold values that do not change during the program's execution.

        
julia> pi
π = 3.1415926535897...
            
julia> pi = 3
ERROR: cannot assign a value to imported variable Base.pi from module Main
  • Variables are mutable
  • Constants are immutable
Datatypes and Operations

Julia provides a complete collection of basic arithmetic and bitwise operators across all of its numeric primitive types, as well as providing portable, efficient implementations of a comprehensive collection of standard mathematical functions.

Expression Name Description
+x unary plus the identity operation
-x unary minus maps values to their additive inverses
x + y binary plus performs addition
x - y binary minus performs subtraction
x * y times performs multiplication
x / y divide performs division
x ÷ y integer divide x / y, truncated to an integer
x \ y inverse divide equivalent to y / x
x ^ y power raises x to the yth power
x % y remainder equivalent to rem(x,y) 

julia> sum = 5 + 3
8
    
julia> modulo = 5 % 4
1

The above code performs an arithmetic addition operation and modulo operation and assign the values to the variables `sum` and `modulo'.

Control Statements

Control statements in Julia allow you to alter the flow of a program based on conditions and loops.

The most common control structures include `if`, `else`, `for`, and `while` loops. For example:


julia> for i = 1:2, j = 3:4
            println((i, j))
            i = 0
       end
(1, 3)
(1, 4)
(2, 3)
(2, 4)

In the example above, the code will check if the value of `x` is positive before printing the message.
